# üöÄ Usman Tools Bot
**Advanced WhatsApp Automation & Research Toolkit**  
**Owner: Usman Tools**

Usman Tools Bot is a modular, multi-feature WhatsApp automation panel built on top of Baileys (WhatsApp Web MD API).  
It is designed for research, automation development, and advanced scripting use-cases.

With Usman Tools Bot you can automate messaging, broadcast, manage groups, create tools and extend the bot using modules.  
The project is lightweight, fast and fully modular.

## üöÄ Installation Guide (Termux)

Follow these step-by-step commands to deploy the bot:

### Step 1: Update & Install Required Packages
```bash
pkg update && pkg upgrade -y
pkg install nodejs -y
pkg install git -y
pkg install ffmpeg -y
pkg install libwebp -y
pkg install imagemagick -y
```

> Tip: If prompted with "Do you want to continue? [Y/n]", type y and press Enter. ‚úÖ

---

Step 2: Clone Repo & Setup
```bash
git clone https://github.com/spabdullahyt-create/Usman-Red-Cup
cd Usman-Red-Cup
termux-setup-storage
```
Step 3: Install Node Modules
```bash
npm install
```
Step 4: Clean Hidden Auth Files
```bash
rm -rf auth_info/*
```
Step 5: Run The Bot
```bash
npm start
```
